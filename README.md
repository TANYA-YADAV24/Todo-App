# Todo-App

A full-stack productivity tool featuring custom authentication, secure session handling, and a clean UI. Built with a modular architecture for easy scalability.

A secure Task Management System built with Flask Blueprints. Features a modular authentication system, session-based login, and flash messaging for real-time user feedback.
Modular To-Do application using Python Flask. Implementation includes Blueprint-based routing, session management, and CRUD functionality.


Flask Task Manager >>>

A modular and secure Todo application built with Python and the Flask web framework. This project demonstrates a professional directory structure using Flask Blueprints to separate authentication logic from main application features.


Features >>>

Modular Architecture: Uses Flask Blueprints for clean code organization.
Secure Authentication: Session-based login and logout system with hashed-like credential checking.
User Feedback: Integrated Flask Flash Messaging for success and error notifications.
Session Management: Secure client-side sessions to track logged-in users. 

Tech Stack >>>

Backend: Python 3.x, Flask
Frontend: Jinja2 Templates, HTML5, CSS3
Security: Flask-Session Management
